SmackAt
=======

Try it out
..........
Visit http://smack.at enter your name and play

Browser Compatibility
..........
Right now it's intended for Chrome. More browsers are on the way.

Installation
............
	$ git clone git://github.com/charlie-rudenstal/SmackAt.git
	$ cd SmackAt
	$ npm install

	And start it with

	$ node server.js
